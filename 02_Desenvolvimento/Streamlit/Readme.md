Podemos criar um app no Streamlit da seguinte forma:

1) Criamos as tabelas no SnowFlake;
2) Criamos as interfaces de Dados dentro do SnowFlake;
3) Criamos um repositório no GitHub para essa novo APP
4) Extraimos o arquivo streamlit_app.py e environment.yml, copiando o conteúdo do environment para um txt (requirements.txt)
5) Entramos no site do Streamlit para lançar o app e em seguida conectamos o GitHub no repositório correto para que ele traga os arquivos que precisaremos.
6) Devemos acoplar o usuário do SnowFlake para que a conta seja acessada
7) Pronto! Temos um app novo em folha.
